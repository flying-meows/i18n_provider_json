## How to run  
`$flutter pub get`  
`$flutter doctor`  
`$flutter devices`  
`$flutter build ios`  
`$flutter run`  

### for web 
`$flutter channel beta`  
`$flutter upgrade`  
`$flutter create .`  
`$flutter build web` 
`$flutter run -d chrome`
